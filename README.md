# vn-test

[![License MIT][license-image]](LICENSE)

Test work for videonow.

## Usage
```bash
$ cd vn-test
$ npm install
$ npm start
```
Open http://localhost:3000 .

Run
```js
localStorage.vnOptions = ``;
```
in console to clear local cached links.
